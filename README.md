# ASB Membership Management System

A streamlined web-based solution for managing community memberships, subscription tiers, and digital identification. This project provides an end-to-end workflow for registering members, managing monthly subscriptions, and issuing verifiable digital ID cards.

## 🚀 Live Demo
You can view the integration of this system and the organization it supports at:
[https://aashsanctuary.gt.tc/](https://aashsanctuary.gt.tc/)

## ✨ Key Features
* **Tiered Membership Model**: Built-in support for multiple subscription levels, including:
    * **Basic Tier**: Essential access and standard community benefits (500).
    * **Elite Tier**: Premium membership with enhanced support and features (1500).
* **Digital Member IDs**: Automated generation of unique member identification numbers.
* **QR Code Verification**: Every digital card includes a unique QR code for instant status verification and authenticity checks.
* **Dynamic Card Generation**: Uses SVG-based templates to ensure high-quality resolution for both digital display and physical printing.
* **Subscription Tracking**: Management system for monthly contributions and membership validity.

## 🛠 Tech Stack
* **Language**: PHP
* **Database**: MySQL / SQL
* **Frontend**: HTML5, CSS3, JavaScript
* **Assets**: SVG for digital membership card templates

## 📦 Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/asb-membership.git
    ```
2.  **Database Setup**:
    * Import the provided `.sql` file into your MySQL database.
    * Configure your database credentials in the configuration file.
3.  **Environment Configuration**:
    * Ensure your server has PHP 7.4+ installed.
    * Adjust the directory permissions for generated QR codes/IDs if necessary.
